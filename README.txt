Este proyecto creare una calculadora utilizando html, css, javascript.

En html empleare los data-value, el cual almacenara la información del los numeros y valores de nuestra
calculadora, y estos data values, serán los que vamos a recibir en javascript.

Por la parte de css estaré utilizando css grid para hacer el diseño de la calcladora, y darle sus propias filas y columnas
que tienen las calcladoras.

Por la parte de javascript una vez que reciba los data values de nuestro html, tendremos una funcion em dichos botones, donde
nos mandará la acción de estos mismos y tendremos una serie de comparaciones.
